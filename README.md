# 🧮 Java Calculator

A console-based calculator built in Java that supports basic arithmetic operations, greatest common divisor (GCD), and digit sum calculation.

## Features

- **Basic operations:** Addition, Subtraction, Multiplication, Division
- **GCD (Greatest Common Divisor):** Uses the Euclidean algorithm
- **Digit Sum (Quersumme):** Calculates the sum of all digits in a number
- **Input validation:** Rejects invalid operations before asking for numbers
- **Division by zero protection:** Handles edge cases gracefully
- **Loop mode:** Keeps running until the user decides to stop

## How to Run

1. Make sure you have **Java 17+** installed
2. Compile the program:
   ```
   javac Main.java
   ```
3. Run it:
   ```
   java Main
   ```
